<!-- GETTING STARTED -->
## Getting Started


1. Clone the repo
   ```sh
   git clone https://github.com/AdamSkrzypinski/warehouse-app-front.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run 
   ```sh
    react-scripts start
   ```
